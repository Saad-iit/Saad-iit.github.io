CareMatch

My Capstone Project to Assist Elders.

For this capstone project, our team of five worked every sprint and rotated roles. One week I focused on frontend design, the next on backend development, and later on security and infrastructure. This rotation gave each of us hands on experience across the entire stack, while keeping security and automation in mind from the start, following a DevSecOps approach.

Our main goal was to design, deploy, and automate our own cloud environment. We built this environment on Proxmox and used tools like Packer and Terraform to create reproducible virtual machine templates for a 3-tier application. Vault was used to manage secrets securely and Vagrant helped replicate the environment locally for development and testing.

On the application side, we developed a backend API connected to an SQL database to store and manage transferred data, as well as others for things such as Google Oauth login. The frontend included responsive CSS to ensure it looked and worked great on all devices. We also built a secure login process, all while hosting the app in the custom cloud infrastructure we had set up earlier.

This project helped me apply a lot of what I had learned, from spinning up cloud infrastructure, to coding the application and ensuring it could be securely deployed with minimal manual intervention. It combined technical challenges, teamwork, and the satisfaction of seeing a fully functional system running in an environment we built ourselves.

The Cost of Connectivity

Understanding IoT Risks and How to Reduce Them.

In this research paper, I explore the rapidly growing world of Internet of Things devices, from what they are and how they work, to the hidden security challenges that come with them. I examine the ways attackers can breach IoT devices, what types of data are most at risk, and how those threats impact both individuals and businesses.

The paper then shifts focus to enterprise environments, outlining policies and best practices to mitigate IoT related risks. I talk about established cybersecurity frameworks, such as the NIST Cybersecurity Framework, to show how organizations can strengthen defenses and prepare for a range of attacks, whether through technical exploits or social engineering.

Before concluding, I address how IoT devices can still be used effectively, if we remain aware of the risks and take security measures. This work highlights why IoT security matters to everyone, even those who do not think they rely on smart technology and why we should be mindful of securing these devices in both our home and business settings.

Ghost of a Controller

Controlling gaming consoles via social media.

Inspired by community driven experiences like Twitch Plays Pokémon, I developed a Python program that lets users control a Nintendo Switch/Switch 2 remotely through popular platforms such as Twitch, YouTube, or Discord.

The solution leverages a Raspberry Pi acting as a Bluetooth HID, sort of like a virtual game controller connected wirelessly to the Switch. The Python app monitors chat inputs or messages on the chosen platform, interprets specific commands corresponding to controller buttons, and sends those signals to the Switch in real time.

This project helped me further learn about hardware emulation, real time data processing, and community engagement to push gaming beyond traditional boundaries, allowing people from all over the world to play together.

Introduction to iPhone Forensics

A Beginner’s Guide to iPhone Forensics.

This research paper walk through an introduction to digital forensics with a focus on the investigative process for iPhones. It explores two major sources of evidence, configuration files and Preinstalled and third party applications.

Configuration files can reveal far more than expected including deleted notifications, pairing certificates, stored passwords, and system logs. Applications, whether preinstalled or downloaded, can also provide critical insights such as WhatsApp messages, social media login history, visited websites, cached downloads from cloud storage services, and more.

By examining these sources, investigators can uncover a detailed picture of device usage and compile stronger more accurate reports. This paper highlights the depth of information accessible through forensic analysis and the importance of using all available resources in a forensic investigation.

BuzzHunt

An OSINT Project.

Developed a Python based application designed to extract comments from popular social media platforms, specifically YouTube and Twitter. The core functionality involves scraping comments and exporting them into a structured CSV file format for easy data manipulation and analysis.

The project features a graphical user interface (GUI) built with Tkinter, making the tool accessible to users without requiring programming knowledge. Through the interface, users can specify the target platform, any specific words or sentences that should be targeted, initiate the scraping process, and view the results directly within the app.

This project helped me better learn skills in Python, API integration, data processing, and user interface design, combining them to create a practical tool for social media data analysis.
